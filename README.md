# Excel-Data-Analysis-Dashboard

## Project Objective

To design an interactive and visually engaging **Global Economic Insights Dashboard** using Microsoft Excel to analyse and compare economic indicators across different countries and regions

## Dataset Used

[Excel-Data-Analysis-Dashboard](https://github.com/emmaloaded7/Excel-Data-Analysis-Dashboard/blob/main/Countries_region.xlsx)

## KPI Questions
* What economic patterns can be indentified across regions?
* Do developed countries consistently have high wages?
* How does population size affect GDP performance?
* Which regions show the best balance between wages and unemployment?
